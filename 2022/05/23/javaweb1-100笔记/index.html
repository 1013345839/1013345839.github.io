javaweb1-100笔记

    mysql服务启动：

cmd,services.msc

             管理员cmd,net start/stop mysql

登录：

            mysql -uroot -p（本地）

            mysql -hip -u -p

退出：

            exit/quit

目录结构：

            mysql安装目录

                    配置文件 my.ini

            数据目录：C:\ProgramData\MySQL\MySQL Server 5.5\data

sql通用语法

            每条 SQL 语句的末端使用分号

            单行注释：– （）含空格

                                #

            多行注释：/* */

sql分类：

            DDL（操作数据表、库）

            DQL（查询表中数据）

            DML（增删改表中的数据）  

            DCL（授权）

DDL

            操作数据库：CRUD

                    Create：create databases 名称；

                                    create databases if not exists 名称；

                                    create databases 名称 character set 字符集；

                                    先判断

                    Retriece：查询 show databases;

                                       查询字符集 show create databases 名称；

                    Update：修改字符集 alter databases 名称 character set 字符集；

                    Delete: drop databases 名称

                                 drop databases if exists 名称；

                    使用数据库：查询当前正在使用的数据库：select databases();

                                            use 名称；

            操作表：

                    C：create table 表名（

                                        列名1 数据类型1，

                                        列名2 数据类型2，

                                        …

                                        列名 数据类型

                                ）；

                            数据库类型：https://www.runoob.com/sql/sql-datatypes-general.html

                                                varchar 字符串类型

                        复制表： create table 表名 like 表名；

                    R：查询所有表名： show tables;、

                            查询表结构：desc 表名；

                    U：修改表名： alter table 表名 rename to 新表名；

                            修改表的字符集：alter table 表名 character set 字符集；

                            添加一列：alter tbale 表名 add 列名 数据类型；

                            修改列名称 类型：alter tbale 表名 change 列名 新列名 新数据类型；

                                                            alter tbale 表名 modify 列名 新数据类型；

                            删除一列：alter table 表名 drop 列名;

                    D：drop table (if exists) 表名;

DML:

            添加数据：insert into 表名（列1，列2…） values(值1，值2 ….)；

                               除数字外用引号

             删除数据：delete from 表名 where 条件；

                                如果不加条件删除所有表中数据

                                truncate table 表名；(先删除表再创建一个一样的空表，效率更高)

            修改数据：update 表名 set 列名1 = 值1，…. [ where 条件]；

                                不加条件则会将表中所有记录全部修改

DQL：

            语法：

                    select

                            字段列表

                    from

                            表名列表

                    where

                            条件列表

                    group by

                            分组字段

                    having

                            分组之后的条件

                    order by

                            排序

                    limit

                            分页限定

            基础查询：

                    多个字段的查询

                            select 字段1，字段2 from 表名；

                            select * from 表名；

                    去除重复的结果集

                            select distinct 字段名 from 表名；

                     as 起别名（可空格）

                    计算列

                            ifnull(表达式1，表达式2) – 替换值

             条件查询：

                    运算符： > < <= >= = <> !=

                                    between .. and … (在2值之间，包括头尾)

                                    like’张%’ 模糊查询    

                                                _    任意一个字符

                                                %    任意多个字符

                                    is null / is not null 不能写=null

                                    and && or || not !    

            排序查询：

                    order by 排序字段1 排序方式1 ，… – 默认升序

                    ASC：升序

                    DESC： 降序

            聚合函数：

                    进行列的计算（排除null的值）

                    count:计算个数

                    max、min、sum、avg

                    例：

                    select count(字段名) from 表名；

                    select count(ifnull(字段名,0)) from 表名；

                    select count(*) from 表名；

            分组查询：

                     group by 分组字段；

                    注意：分组之后查询的字段：分组字段、聚合函数

                    where 和having 的区别：

                            where 在分组前限定，having在分组后限定

                            where后不跟聚合函数，having 可以

            分页查询：

                    limit 开始的索引，每页查询的条数；

                    开始的索引 = （当前页码 - 1）*每页显示的条数

DCL： 管理用户，授权

约束

            主键约束：primary key 含义：非空且唯一

                    创建表时添加主键 数据类型后加primary key

                    删除主键 alter table 表名 drop primary key;

                    创建完后添加 alter table 表名 modify 字符名 数据类型 primary key;      

                    自动增长 key 后面加 auto_increment

                                    删除/添加自动增长 alter table 表名 modify 字符名 数据类型（auto_increment）;

           非空约束：not null

                    创建表时添加非空约束（在数据类型后加 not null）

                    创建完表后添加非空约束 alter table 表名 modify 字符名 数据类型 not null；– 删除则不写not null

            唯一约束：unique

                    创建表时添加唯一约束（在数据类型后加 unique）

                    创建完表后添加唯一约束 alter table 表名 modify 字符名 数据类型 unique；

                    删除唯一约束： alter table 表名 drop index 字符名;

            外键约束：foreign key

                    创建表时添加外键：

                    create table 表名（

                                 ….

                                 外键列

                                 comstraint 外键名称 foreign key (外键列名称) refernces 主表名称（主表列名称）

                    ）；

                    删除外键：alter table 表名 drop foreign key 外键名称；

                    添加外键：alter table 表名 add comstraint 外键名称 foreign key (外键列名称) refernces 主表名称（主表列名称）;

                    **设置级联更新(删除)**：alter table 表名 add comstraint 外键名称 foreign key (外键列名称) refernces 主表名称（主表列名称） on update（delete） cascade;

            范式

            

多表查询

            内连接查询：

                    隐式： where …..

                    显式： select 字段列表 from 表名1 [inner] join 表名2 on 条件；

            外连接查询：

                    左外连接： select 字段列表 from 表1left [outer] join 表2 on 条件；

                                        查询左表所有数据以及右表交集部分

                    右外连接： select 字段列表 from 表1 right [outer] join 表2 on 条件；

                    子查询： 查询中嵌套查询

                                1.子查询的结果单行单列

                                        可作为条件使用运算符判断

                                2.子查询结果多行单列

                                        使用运算符in判断

                                3.多行多列

                                        子查询可以作为一张虚拟表

事务：

            开启事务：start transaction;

            回滚：rollback;

            提交：commit;

            原子性、持久性、隔离性、一致性

            隔离级别（暂做了解，忘了网上再查）

JDBC:

            java database connectivity Java数据库连接
